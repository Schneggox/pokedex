# Pokédex

Die App lädt Pokémon-Daten von der [PokéAPI](https://pokeapi.co/) und stellt sie als durchsuchbare Kartenliste dar.

## Features

- 30 Pokémon werden beim Start geladen, weitere per "Load More"
- Klick auf eine Karte öffnet eine Detailansicht mit Stats
- Pfeil-Navigation zwischen Pokémon im Detail-Overlay
- Suchfunktion (ab 3 Buchstaben)
- Hintergrundfarbe je nach Pokémon-Typ
- Loading-Spinner als Feedback beim Laden
- Responsive von 320px bis 1440px
